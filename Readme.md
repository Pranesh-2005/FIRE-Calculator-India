# 🔥 FIRE-Calculator-India

A web application to calculate your **FIRE (Financial Independence, Retire Early)** number tailored for Indian users. Easily estimate how much you need to retire early and live your dream life, with calculations based on Indian cost-of-living and investment returns.

---

## 📖 Introduction

**FIRE-Calculator-India** helps you plan for early retirement by estimating the corpus you need to achieve financial independence in India. With a simple interface and robust backend, this tool incorporates inflation, investments, and Indian financial realities so you can plan your future with confidence.

---

## ✨ Features

- 🇮🇳 **India-specific FIRE calculation**
- 📈 Investment and inflation inputs
- ⚡ Fast, modern web UI
- ♻️ Real-time calculations using FastAPI backend
- 💾 PWA support (Installable on devices)
- 🔒 Data privacy — no personal info stored

---

## 🚀 Installation

#### 1. Clone the repository
```sh
git clone https://github.com/pranesh-2005/FIRE-Calculator-India.git
cd FIRE-Calculator-India
```

#### 2. Backend Setup

- Install dependencies:
    ```sh
    pip install -r requirements.txt
    ```
- Create a `.env` file for your environment variables (if needed).

#### 3. Frontend Setup

- Open the `frontend/index.html` in your browser,
  **OR** serve the frontend using your favorite server.

#### 4. Run the backend

```sh
uvicorn app:app --reload
```

---

## 🖥️ Usage

1. Visit the frontend in your browser (`frontend/index.html` or hosted URL).
2. Enter your details: expenses, investments, expected returns, etc.
3. Click **Calculate**.
4. View your FIRE number and estimated retirement timeline!
